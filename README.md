Работа выполнена в рамках ИКМ по предмету ЯП Python.
Программа считывает данные о квартиросъёмщиках из текстового файла и на их основе формировать следующие отчёты:
1.	Полный список всех квартиросъёмщиков, который будет отсортирован следующему ключу: количество прописанных человек (по убыванию) + адрес (по возрастанию). 
2.	Список всех квартиросъёмщиков, имеющих льготы, отсортированный по следующему ключу: этаж (по возрастанию) + количество прописанных человек (по убыванию) + общая площадь (по возрастанию). 
3.	Список всех квартир с общей площадью в диапазоне от N1 до N2 кв.м., отсортированный по следующему ключу: наличие льготы (по возрастанию) + общая площадь (по убыванию).

Интерфейс простой, интуитивно понятный, с защитой от ошибок ввода
Программа запускается из основного файла "mymain.py"
